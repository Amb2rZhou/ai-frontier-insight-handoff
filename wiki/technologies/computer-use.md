## Key Developments

### Landmark Launches (W14 — "Agent Moved into the OS")
Claude Computer Use launched for Mac (Mar 23-24), using a layered decision tree: API connectors first, browser automation second, screen control as fallback. For Pro ($17/mo) and Max ($100-200/mo) subscribers. Gemini Screen Automation deployed on Galaxy S26 and Pixel 10 with sandbox design. OpenAI Codex gained plugin system for direct enterprise tool access.

### Evolution
Perplexity launched "Computer" unified AI agent system (Feb 26). Anthropic acquired computer-use AI startup Vercept (Feb 26). OpenAI Codex upgraded into full computer operator with 90+ plugins (Apr 17). Grok 4.3 beta added multi-format file creation (Apr 18).

### Research
Study revealed safety vulnerabilities in computer-use agents from benign instructions (Apr 17). Salesforce proposed GPA for GUI Process Automation from demonstrations (Apr 6).

### Strategic Significance
Computer use represents the paradigm shift from "advisor" to "operator." It changes AI competition from "who gives the best answer" to "whose agent lives on your device first."

## Weekly Insights
- **W13 (Water Sellers' Water Getting Cheaper)**: Claude Computer Use launched alongside the OpenClaw ecosystem explosion — agent adoption accelerated but model providers found themselves squeezed into commodity pricing as the value shifted toward the execution environment.
- **W14 (Agent Moved into the OS)**: Four companies (Anthropic, Google, OpenAI, xAI) simultaneously crossed from "advisor" to "operator" in one week — Claude controlling Mac, Gemini operating phone apps, Codex gaining enterprise plugins. This formally broke the chatbox paradigm and established "whose agent lives on your device first" as the new competitive frontier, with first-mover advantage driven by accumulated user context rather than benchmark scores.
- **W17 (Context War)**: Codex upgraded to full computer operator with 90+ plugins, confirming that computer use has become table stakes — the competitive differentiator is now the context layer (persistent memory, user history, workflow knowledge) that agents accumulate through sustained device-level interaction.
