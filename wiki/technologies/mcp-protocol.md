# MCP Protocol (Model Context Protocol)

An open protocol for connecting AI models to external tools and data sources. Originally developed by Anthropic, now becoming an industry standard for agent-tool interoperability.

## Key Developments

### Adoption
Mobile-MCP enabled LLMs to autonomously discover Android app capabilities (Feb 26). The protocol became central to agent framework interoperability discussions.

### Ecosystem
MCP became a standard integration pattern for AI agents connecting to external services. Enterprise adoption accelerated as agents needed standardized tool access.

## Weekly Insights
- **W12 (Model Race to Habitat War)**: Manufact raised $6.3M to build on MCP as the standard connector between agents and tools — positioned as "USB-C for AI agents," a cross-platform antidote to the proprietary platform lock-in being constructed by Microsoft, Google, and NVIDIA.
- **W17 (Context War)**: MCP became the protocol of choice for incumbents opening their platforms to agents — Salesforce Headless 360 exposed 60+ MCP tools, SAP added full MCP support, ServiceNow's AI Agent Fabric used MCP alongside A2A. The protocol is transitioning from a developer standard to enterprise infrastructure.
