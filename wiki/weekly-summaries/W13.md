# W13: The Water Sellers' Water Is Getting Cheaper

**Theme:** Value Chain Redistribution
**Period:** 2026-03-17 to 2026-03-23

## Summary

The model layer's pricing power eroded from two directions simultaneously. Top applications started building their own models: Cursor based Composer 2 on Moonshot's open-source Kimi, achieving frontier-level coding at a fraction of API costs. Meanwhile, NVIDIA's NemoClaw and Nemotron-Cascade 2 made local agent deployment viable.

OpenAI launched GPT-5.4 mini/nano at dramatically lower prices — not from generosity, but to delay customer migration to open-source alternatives. Agent adoption was exploding (OpenClaw ecosystem, Claude Computer Use launched), but model providers were being squeezed into commodity pricing.
