# W12: From Model Race to Habitat War

**Theme:** Infrastructure Power Shift
**Period:** 2026-03-10 to 2026-03-16

## Summary

Model-level competitive moats collapsed as NVIDIA open-sourced a 120B agent model, LeCun bet $1B on post-LLM architectures, and Chollet declared the next breakthrough needs a new paradigm. Competition shifted to platform control: Microsoft embedded agents in Office 365, Google made Chrome native for agents, VS Code launched full Autopilot mode, NVIDIA released enterprise agent platform.

The key dynamic: when agent capability becomes commoditized infrastructure, whoever controls the execution environment controls user migration costs. ByteDance's Seedance 2.0 compliance issues proved that without safety infrastructure, products can't even enter the market.
