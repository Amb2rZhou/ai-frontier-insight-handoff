# W15: The Math Doesn't Work

**Theme:** Economic Model Restructuring
**Period:** 2026-03-31 to 2026-04-06

## Summary

After agents moved into the OS, the bill arrived. Agent sessions consume 50-100x more tokens than conversations. Anthropic restricted third-party tool access citing "disproportionate infrastructure pressure." OpenAI reset all Codex rate limits. GitHub API rate limits became an agent scaling bottleneck. Reddit analysis showed agent frameworks waste 350K+ tokens per session.

Heavy agent users cost 100-1000x more than chat users but pay the same subscription. The subscription model's mathematical foundation collapsed. Meanwhile, Google released Gemma 4 under Apache 2.0 — a weekend project could now build a complete local agent, and the tools to bypass the "agent tax" matured in the same week the tax was imposed.
