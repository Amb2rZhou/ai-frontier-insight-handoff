# W14: Agent Moved into the Operating System

**Theme:** Interaction Paradigm Shift
**Period:** 2026-03-24 to 2026-03-30

## Summary

The chat-box paradigm was formally broken. In one week: Anthropic launched Claude Computer Use (AI controls Mac), Google Gemini gained device-level agent capability on phones, Google released Gemini 3.1 Flash Live for real-time voice+vision agents, and OpenAI Codex gained plugin system for enterprise workflows.

Four companies simultaneously crossed from "advisor" to "operator." When agents can directly operate devices and data, "whose agent lives on your device first" becomes more critical than "whose model benchmarks highest." The litellm supply chain attack and Anthropic model security concerns reminded us: we're handing over the keys, but the locks haven't been built yet.
