## Key Developments

### Ecosystem Growth
Ecosystem exploded with global meetups and tooling (Feb 25). Framework update gained high engagement (Mar 30). Added native video generation with multi-provider support (Apr 6). Added dreaming feature and hardened security (Apr 9). Tencent and OpenClaw collaborated on agent harness performance (Apr 21).

### Benchmarking
InternLM released WildClawBench for in-the-wild agent evaluation (Mar 29). ClawBench presented real-world evaluation across 144 platforms showing best model (Claude Sonnet 4.6) at 33.3% — roughly half of sandbox scores (Apr 12).

### Community

## Weekly Insights
- **W12 (Model Race to Habitat War)**: Received engineering support from both NVIDIA and Microsoft — seemingly contradictory for companies building their own agent platforms, but strategically rational: participating in open-source standards during the formative phase is more valuable than resisting them, just as Google supported Android.
- **W13 (Water Sellers' Water Getting Cheaper)**: The OpenClaw ecosystem explosion drove massive API call volume but contributed to the cost crisis — agent frameworks wasting 350K+ tokens per session on redundant context highlighted that the real bottleneck is not model capability but agent efficiency.
- **W15 (The Math Doesn't Work)**: Anthropic restricted Claude subscribers' access to OpenClaw, citing infrastructure pressure — the first time a model provider penalized its most engaged agent ecosystem. Alibaba's Qwen immediately positioned OpenClaw compatibility as a selling point, demonstrating how restriction creates competitive opportunity.
- **W16 (No One Is Verifying)**: ClawBench revealed real-world agent success rates (33.3% for best model across 144 platforms) are roughly half of sandbox scores — providing the first rigorous evidence that existing benchmarks dramatically overestimate agent production readiness.
